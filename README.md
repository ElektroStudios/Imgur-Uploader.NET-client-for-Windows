## 👋 Introduction

This project is a small, straightforward Windows desktop client built in VB.NET using WinForms. It offers a humble, lightweight graphical interface designed to help you upload images directly to your personal Imgur account using your Imgur API.

## 💡 Motivation

Uploading local images to Imgur usually requires opening a web browser, navigating to the site, dragging files onto the page, and waiting for a bulky web layout to load just to get a simple URL. When you need to host images quickly for forums, markdown documentation, or sharing scripts, this process feels slow. I wanted to build a tiny, Windows tool to bypass the browser overhead entirely, letting you upload images directly from your desktop in a single step.

##### ⚡ The Real Question
###### Why waste time opening a full browser tab and dealing with a heavy website just to get an image link, when you can upload your files directly from your desktop in a second?.

## 🚀 Getting Started

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/Imgur-Uploader.NET-client-for-Windows/releases/latest)**.
2. Download the latest `.zip` archive or the `.exe` setup installer, depending on your preference.
3. If you downloaded the `.zip` archive, extract its contents to your preferred directory.
   <br> 
   If you downloaded the `.exe` file, run it and follow the installation wizard.
4. Run the executable file to launch the application.

### API key Registration

To use Imgur Uploader.NET, you need a Imgur's API key. To request your API key follow the instructions on this link:

 - https://apidocs.imgur.com/#register

### API key configuration

Once you have requested your API key on Imgur's website, just open the **ImgurUploader.ini** file and fill the missing values with your settings:

    [API]
    ClientId=(Your client Id)
    ClientSecret=(Your client secret)
    
These steps are necessary in order to use Imgur Uploader.NET.
